# Hamilton Helmer Moat Assessment

## Purpose
Evaluate each competitor's competitive moat using the 7 distinct sources of competitive advantage identified by Hamilton Helmer (7 Powers).

## The 7 Moats

### 1. Scale Advantages
- Fixed costs spread over larger volume
- Distribution scale (more stores, more routes)
- Manufacturing scale (lower per-unit cost)
- **Score**: Strong / Moderate / Weak / Nascent

### 2. Network Effects
- Value increases as more users join
- Direct (marketplaces) vs. indirect (platforms)
- Two-sided networks (buyers + sellers)
- **Score**: Strong / Moderate / Weak / Nascent

### 3. Switching Costs
- Cost (time, money, effort) to switch to alternative
- Technical switching (data migration, retraining)
- Procedural switching (workflow disruption)
- Financial switching (contract penalties, sunk costs)
- **Score**: Strong / Moderate / Weak / Nascent

### 4. Brand
- Premium pricing power from brand recognition
- Trust transfer (brand extends to new products)
- **Score**: Strong / Moderate / Weak / Nascent

### 5. Regulatory
- Licenses, permits, patents that create barriers
- Regulatory capture / favorable regulation
- **Score**: Strong / Moderate / Weak / Nascent

### 6. IP / Proprietary Technology
- Patents, trade secrets, proprietary algorithms
- Technical complexity that's hard to reverse-engineer
- **Score**: Strong / Moderate / Weak / Nascent

### 7. Distribution
- Exclusive or dominant distribution channels
- Shelf space, partner relationships, app store ranking
- **Score**: Strong / Moderate / Weak / Nascent

## HYPERION Application
The Competitive Intelligence agent (Agent 4, STANDARD tier) uses this framework to assess moats for the top 5 competitors. It doesn't just list moats — it identifies which are **defensible** (getting stronger) vs. **eroding** (getting weaker). This distinction is critical for the Synthesis Lead's recommendation.
