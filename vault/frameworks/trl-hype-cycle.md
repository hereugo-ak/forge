# Technology Readiness Levels (TRL) & Gartner Hype Cycle

## Purpose
Assess emerging technologies on the NASA TRL scale (1-9) and plot on the Gartner Hype Cycle to separate hype from production-ready tech.

## TRL Scale

| Level | Label | Description |
|---|---|---|
| 1 | Basic Research | Basic principles observed and reported |
| 2 | Applied Research | Technology concept and/or application formulated |
| 3 | Proof of Concept | Analytical and experimental critical function proof |
| 4 | Lab Validation | Component/subsystem validation in laboratory |
| 5 | Field Validation | Component validation in relevant environment |
| 6 | System Demo | System/subsystem model or prototype demonstration |
| 7 | System Prototype | Prototype in operational environment |
| 8 | System Complete | Actual system completed and qualified through test |
| 9 | Deployed | Actual system proven through successful operations |

## Gartner Hype Cycle Phases

1. **Innovation Trigger**: Breakthrough, public demonstration
2. **Peak of Inflated Expectations**: Over-enthusiasm, unrealistic projections
3. **Trough of Disillusionment**: Interest wanes, experiments don't deliver
4. **Slope of Enlightenment**: Second-gen products, best practices emerge
5. **Plateau of Productivity**: Mainstream adoption starts

## Assessment Template

| Technology | TRL | Hype Cycle Phase | Years to Maturity | First-Mover Advantage? |
|---|---|---|---|---|
| [Tech 1] | 1-9 | Phase name | X years | Yes/No/Risky |
| [Tech 2] | 1-9 | Phase name | X years | Yes/No/Risky |

## HYPERION Application
The Innovation Analyst (Agent 13, STANDARD tier) uses Wayback Machine to track hype vs. reality over time. It distinguishes between technologies at the slope of enlightenment (TRL 8, ready for production) vs. peak of inflated expectations (TRL 4, 2-3 years from production). It always assesses first-mover advantage — in some markets, being first is a disadvantage.
